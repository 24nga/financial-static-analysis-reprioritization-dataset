# Financial Static Analysis Reprioritization Dataset

This dataset supports research on:

**Risk-Aware Static Analysis Reprioritization using On-Premise LLMs for Human-AI Code Review Governance in Financial Systems**

## Key Features
- 233-rule financial inspection framework
- Synthetic financial-domain code (AML, FRTB, ALM, Lending, Accounting, Credit)
- Context-aware priority labels (fixed vs expert vs LLM)
- Actionability and impact annotations
- Designed for safe auto-approval and decision-making research

## Research Gap Addressed
Existing datasets focus on:
- vulnerability detection
- code repair

This dataset focuses on:
→ **decision-making (what to fix first, what to auto-approve)**
